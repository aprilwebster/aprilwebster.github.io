A little bit about me

my day job

software engineer, data scientist, statistician

I'm a software engineer on the IBM Watson team building tooling to help developers train and evaluate ML models. I develop applications using Node.js, React, Python and Java.

I've also worked as a data scientist, a data analyst and a statistics consultant. A common theme across all these roles has been data, particularly on how to use and visualize it to help users make more informed decisions.

some of my projects

a selection of publicly available projects

  • Tone Analyzer for Customer Engagement - a web application demonstrating how the Watson Tone Analyzer Customer Care endpoint can be used.
  • Food Coach - a chatbot that integrates Watson Tone Analyzer and Watson Conversation Services.
  • Watson Developer Cloud SDKs - I contributed a Watson Tone Analyzer and Conversation integration example to the SDKs.
  • IBM Content and Predictive Analytics (ICPA) for Healthcare - an NLP analytic workflow system for the healthcare domain. I contributed to the research prototype that was productized as ICPA.
